# 📸 Telegram EXIF Spoofer Bot

Красивый и удобный Telegram-бот для обработки фотографий: удаление реальных EXIF-данных, подмена GPS-координат, модели устройства и уникализация изображения (незаметно для глаза, но уникально для компьютера).

## Возможности

- ✅ Полное удаление оригинальных EXIF (включая GPS, серийные номера и т.д.)
- 📍 Подмена координат на случайные из большой базы (~500 точек, в т.ч. Россия, Арктика и др.)
- 📱 Случайная модель телефона из популярных брендов: **Apple, Samsung, Google, Xiaomi**
- 🧠 Генерация реалистичных EXIF-тегов (Exposure, ISO, FocalLength, Software и др.), как у настоящего снимка с телефона
- 🔄 Уникализация фото: лёгкое изменение пикселей через ресайз (глазу незаметно, хеш и perceptual hash меняются)
- 💬 Красивый интерфейс с эмодзи, понятными сообщениями и мгновенной обработкой
- 🚀 Готов к деплою на Railway (polling)

## Как использовать

1. Добавьте бота в Telegram
2. Отправьте любое фото (как фото, не документ)
3. Получите обработанную версию с новым EXIF и уникальными пикселями
4. Готово! Теперь фото "как с другого телефона" и с другой локации.

## Установка и запуск

### Локально

```bash
git clone <your-repo>
cd telegram-exif-bot
python -m venv venv
source venv/bin/activate  # или venv\Scripts\activate на Windows
pip install -r requirements.txt
cp .env.example .env
# отредактируйте .env и вставьте токен от @BotFather
python bot.py
```

### На Railway (рекомендуется)

1. Fork / Push этот репозиторий на GitHub
2. Зайдите на [Railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Выберите репозиторий
4. В Variables добавьте:
   - `TELEGRAM_BOT_TOKEN` = ваш токен
5. Deploy
6. Бот запустится автоматически (polling)

> **Примечание**: На бесплатном плане Railway сервис может засыпать. Для 24/7 используйте платный план или настройте webhook (см. ниже).

## Настройка Webhook (для продакшена)

Раскомментируйте секцию webhook в `bot.py` и добавьте переменную `RAILWAY_PUBLIC_DOMAIN` или используйте `https://<your-app>.up.railway.app`.

## Кастомизация

- Хотите больше брендов или конкретные модели? Добавьте строки в CSV в папке `data/`
- Хотите фиксированные координаты или свой набор? Замените `coords.csv`
- Улучшить генерацию EXIF? Правьте функцию `create_fake_exif` в `bot.py`

## Безопасность и приватность

Бот **не сохраняет** ваши фото и данные. Обработка происходит в памяти, после отправки результат удаляется.
